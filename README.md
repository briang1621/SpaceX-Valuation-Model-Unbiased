# SpaceX-Valuation-Model-Unbiased

This repository contains an unbiased SpaceX valuation model developed by Dr. Brian Scott Glassman. The model shows the core drivers of revenue, profitability, and growth and is accompanied by a 40-page article. The author is a former senior mechanical engineer, NYU professor, and holds a PhD in Business Innovation.
